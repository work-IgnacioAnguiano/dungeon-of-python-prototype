# Dungeon of Python Prototype

A polished browser prototype based on coursework on creating a Python dungeon RPG.
The prototype keeps the core ideas from the source project: class selection,
turn-based battles, enemy waves, critical hits, blocking, dodging, potions,
random dungeon events, loot, special moves, and a final Dragon King boss fight.

## How to Run

Open `index.html` in a browser.

No installation is required.

## Controls

- **Attack**: reliable weapon damage
- **Special**: stronger class ability with cooldown
- **Defend**: reduces the next enemy attack
- **Use Potion**: restores health, then the enemy takes a turn

## Prototype Features

- Warrior and Mage hero classes
- Scaling enemy waves
- Elite enemies and final boss
- Health bars and combat stat cards
- Inventory and potion tracking
- Random treasure, trap, and healing events
- Responsive polished layout

## Original Project Notes

The source folder contained several Python iterations during the course as a student:

- `dungeon_game.py`
- `test.py`
- `Character dev.py`
- `backup main.py`
- `previous week work.py`

These files showed the main design direction: a turn-based dungeon crawler with
classes, enemies, specials, inventory, potions, generators, and live battle logs.
This prototype turns those ideas into a focused playable demo.
